About

Hey, I'm Rhea! I'm an undergraduate at Purdue University studying UX Design with minors in Computer Science, Psychology, and Management. My skill set is predominantly in user research, user interface design, and prototyping.

I have experience working with cross-disciplinary teams to design for primarily web products. During my first internship, she helped build the application flow and initial UI patterns, prototype the web UI using Sketch and InVision, and create the visual design for a cybersecurity startup’s minimum viable product.

Currently, I'm interning at Purdue’s Office of Institutional Research, Assessment, and Effectiveness. She is focused on conducting user research to improve Tableau dashboards that track and compare Purdue student performance.

Currently, I'm interested UX internships for spring (remote) and summer 2018. I'm looking for opportunities for cool projects with great teams, so shoot me an email if you'd think I'd be a good fit!
